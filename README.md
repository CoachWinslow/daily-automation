# Daily Automation

An automated daily content delivery pipeline built with Google Apps Script, the Anthropic Claude API, and Slack Incoming Webhooks.

## What It Does

Every morning, two automated triggers fire:

- **5:00 AM**: A personalized motivational briefing with daily action items, health reminders, and career goals is delivered to a private Slack channel (`#enveniamviam`) and Gmail.
- **9:00 AM**: A real-time tech news digest covering SaaS, Cloud Computing, Data Centers, and Chip Makers is generated by Claude with live web search, then delivered to a Slack channel (`#tech-news`) and Gmail.

## Architecture

```
Google Apps Script (Engine)
    |
    |-- callClaude() --> Anthropic Claude API (with web search)
    |                         |
    |                         v
    |                    Returns live tech news
    |
    |-- sendToSlack() --> Slack Incoming Webhook --> #tech-news
    |-- sendToSlack() --> Slack Incoming Webhook --> #enveniamviam
    |-- MailApp.sendEmail() --> Gmail
    |
    |-- Time-driven triggers (5 AM, 9 AM) --> Serverless scheduling
```

## Tech Stack

- **Language**: JavaScript (Google Apps Script / ES6)
- **AI**: Anthropic Claude API (claude-sonnet-4) with web search tool
- **Messaging**: Slack Incoming Webhooks
- **Email**: Gmail via Google Apps Script MailApp
- **Secrets Management**: Google Apps Script Script Properties
- **Scheduling**: Google Apps Script time-driven triggers (serverless)

## Functions

| Function | Purpose |
|----------|---------|
| `sendToSlack(webhookUrl, message)` | Reusable helper that POSTs a JSON payload to any Slack webhook URL |
| `callClaude(prompt)` | Sends a prompt to the Claude API with web search enabled, returns text response |
| `morningTechNews()` | Generates daily tech news via Claude, fans out to Slack and Gmail |
| `morningMotivation()` | Delivers daily motivation and reminders to Slack and Gmail |
| `testSlack()` | Smoke test for both Slack webhook connections |
| `testClaude()` | Smoke test for Claude API connection |

## Secrets Management

All credentials are stored in Google Apps Script Script Properties (never hardcoded):

- `SLACK_WEBHOOK_TECHNEWS`: Webhook URL for #tech-news channel
- `SLACK_WEBHOOK_ENVENIAMVIAM`: Webhook URL for #enveniamviam channel
- `CLAUDE_API_KEY`: Anthropic API key

This follows the same principle as AWS Systems Manager Parameter Store / Secrets Manager.

## Setup

1. Create a new Google Apps Script project at [script.google.com](https://script.google.com)
2. Copy `Code.gs` into the project
3. Add your secrets to Project Settings > Script Properties:
   - `SLACK_WEBHOOK_TECHNEWS`
   - `SLACK_WEBHOOK_ENVENIAMVIAM`
   - `CLAUDE_API_KEY`
4. Create two Slack Incoming Webhooks (one per channel)
5. Set time-driven triggers:
   - `morningMotivation` at 5:00 AM
   - `morningTechNews` at 9:00 AM
